Bias–Variance Decomposition ($n=2$)

ค่าคลาดเคลื่อนนอกตัวอย่างเฉลี่ยถูกแยกเป็น bias² และ variance สำหรับแต่ละโมเดล

Target Model bias² variance $E_{\text{out}}$
สังเกต: Linear model กับ $\sin(\pi x)$ มี bias² ต่ำ (~0.21) แต่ variance สูงมาก (~1.67) เพราะเส้นตรงพยายามผ่านแค่ 2 จุดที่สุ่มมา
สังเกต: Constant model กับ $x^2$ ทำงานได้ดีกว่าที่คาด (bias² ต่ำ) เพราะ $x^2$ สมมาตรบน $[-1,1]$ ค่าเฉลี่ย $1/3$ ใกล้เคียงฟังก์ชันจริง

Average Fit Plot

  • เส้นเขียว: ฟังก์ชันเป้าหมาย $f(x)$
  • เส้นประแดง: แบบจำลองเฉลี่ย $\bar{g}(x)$ จาก simulation
  • เส้นสีเทาบาง ๆ: hypothesis จากชุดข้อมูลสุ่ม 20 ชุด
  • ความกว้างของเส้นเทา = variance
  • ระยะห่างเส้นแดงจากเส้นเขียว = bias

Learning Curve Plot

  • เส้นประ: $E_{\text{in}}$ บนชุดฝึก
  • เส้นทึบ: $E_{\text{out}}$ บนข้อมูลใหม่
